Crunchyroll

Redesign

Role User Interface Design
Team Individual
Context HCDE Course Project
Background

For my technical communication course, I had to analyze and redesign an existing product. I chose to analyze Crunchyroll, the leading anime streaming platform, because of my frustrations when using it. The final product of this assignment was a report detailing my analysis and redesign of Crunchyroll. It is available for reading through this link here. However, it is pretty lengthy. Please view a shortened overview of the redesigned artifacts and reasoning below.

Design Process

Ideology

For this project, I was introduced to Jakob Nielsen's usability principles and gestalt principles. I focused on incorporating these principles into my redesign of Crunchyroll. The main problem that I found with Crunchyroll's interface was the lack of consistency across platforms. I attempted to create a consistent visual and design language for Crunchyroll in my redesigns below.

Artifacts

Reflection

This project opened my eyes to Jakob Nielsen's usability principles. I will definitely apply these guidelines to my future designs. I also learned how difficult it is to design for multiple platforms. Crunchyroll had so many different styles for its platforms, possibly due to software restrictions.
